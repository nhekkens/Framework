Tooltips

Example of tooltips.

Demo

Hover the tooltip trigger to see the tooltip working.

There are three attributes you need to add the element you want the tooltip on:

  • data-placement=""
  • data-toggle="tooltip"
  • data-original-title=""

Add one of the four to data-placement="" to change the position of the tooltip:

  • data-placement="top"
  • data-placement="right"
  • data-placement="bottom"
  • data-placement="left"

Change the text in data-original-title="" for a different message in the tooltip.
